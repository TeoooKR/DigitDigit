Game Description

DigitDigit is a fast-paced game that helps you learn the binary system while testing your speed and accuracy.

üéÆ How to Play

  • Your goal is to enter binary numbers sequentially, starting from 1.
  • Press the assigned keyboard keys to toggle each bit to match the target number.
  • Clear the game as fast as possible to set a new record!

  • Timed: Count as many numbers as you can in 10 seconds!
  • Instant: No target, just toggle bits and see the decimal result.
